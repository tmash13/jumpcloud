# Averages

A module that takes in actions as json and calculates average times for each action

## Library

nlohmann json parser is used for json parsing.
The library can be found at https://github.com/nlohmann/json

### Compiling and running

The project was created using Xcode, so install Xcode from the App store and double-click on the average.xcodeproj file to open it.

#### Running in a terminal

Once open, you can press command+b to build the project, then you can browse to where the binary is and run it from the terminal.
The location of the binary file is:
average/build/Debug

#### Running in Xcode

To run in Xcode, press command+r. Once the app is running, you can use the termial window to view the output.
